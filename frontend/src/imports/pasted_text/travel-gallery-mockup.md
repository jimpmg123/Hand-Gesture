Create a desktop web app gallery page mockup for a project called “Travel From Photo”.

Project concept:
This is an AI-powered travel and food memory navigator. Users upload travel photos and food photos. The system helps identify places, connect memories to cities, suggest routes, and extend food images into restaurant discovery. The gallery is a signed-in personal archive where uploaded memories are grouped by city rather than shown as isolated single images.

Design direction:
Use a warm, elegant, editorial dashboard style similar to a modern academic product mockup. The interface should feel minimal, soft, polished, and trustworthy. Use a light cream or off-white background, pale gray or ivory cards, muted teal/green accent colors, rounded corners, subtle borders, and gentle shadows. Typography should combine a strong serif heading with clean sans-serif body text.

Background styling:
Decorate the background subtly with a very faint world map graphic or soft world map silhouette. The map should be low-contrast, elegant, and atmospheric, not distracting. It should look like a soft decorative layer behind the main content. This world map treatment should feel consistent with the Home, Gallery, and Profile pages.

Important spacing requirement:
Make the layout inspired by the provided reference image, but improve spacing and readability. Text should never feel glued to the card edges. Every card, panel, button group, and content block should have generous inner padding. Increase whitespace inside cards, especially around headings, labels, dates, descriptions, and buttons. Keep all content comfortably inset from borders.

Gallery behavior concept:
Each card represents a grouped image collection for a city or city-based memory cluster.
These are not single-photo cards.
Each card is a thumbnail summary for a set of uploaded images from the same city.
For example, multiple uploaded photos from Busan should appear under one Busan group card.
Food and landmark memories from the same city may be grouped together if appropriate.
